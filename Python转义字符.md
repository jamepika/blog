### Python转义字符含义

| 字符   |                             解释                             |
| ------ | :----------------------------------------------------------: |
| \    |                 续行符，用在代码行尾可以续行                 |
| \\   |                        代表反斜杠本身                        |
| \'   |               对单引号进行转义，代表单引号本身               |
| \"   |               对双引号进行转义，代表双引号本身               |
| \a   |        响铃，在cmd终端或terminal终端执行后电脑有响声         |
| \b   |                        退格=Backspace                        |
| \000 |                       四大皆空，输出空                       |
| \n   |                            换行符                            |
| \v   |                          纵向制表符                          |
| \t   |                          横向制表符                          |
| \r   | 将 \r 后面的内容移到字符串开头，并逐一替换开头部分的字符，直至将 \r 后面的内容完全替换完成 |
| \f   |                             换页                             |
| \yyy |     八进制数: y 代表 0~7 的字符，例：\110 代表大写字母H      |
| \xyy | 十六进制数: 以 \x 开头，y 代表的字符，例：\x48 代表大写字母H |
